Завдання 1: Декоратор для вимірювання часу виконання функції
Створіть декоратор, який вимірює час виконання функції. Потім застосуйте його до функції, яка обчислює факторіал числа за допомогою рекурсії.

Завдання 2: Ітератор для списку
Створіть клас, який працює як ітератор для списку чисел від 1 до N, де N — число, передане під час ініціалізації. Перебирайте ці числа в циклі for.

Завдання 3: Замикання для створення множинних функцій
Напишіть функцію, яка приймає число n і повертає замикання, що збільшує будь-яке число на n. Тестуйте створення замикань для різних значень n.

Завдання 4: Генератор для парних чисел
Напишіть генератор, який буде генерувати парні числа від 0 до N (де N — число, передане при виклику генератора). Використовуйте цей генератор для обчислення суми всіх парних чисел у діапазоні.

Завдання 5: Декоратор для перевірки доступу
Створіть декоратор, який перевіряє, чи є у користувача доступ до певної функції. Якщо доступ є, функція виконується, якщо ні — виводиться повідомлення "Доступ заборонено".
