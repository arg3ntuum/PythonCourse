Завдання 1: Функція для перевірки парності числа
Напишіть функцію is_even(number), яка приймає ціле число number і повертає True, якщо число парне, і False в іншому випадку.
Використовуйте цю функцію для перевірки парності введеного користувачем числа та виведіть результат.

Завдання 2: Функція для знаходження найбільшого з трьох чисел
Напишіть функцію max_of_three(a, b, c), яка приймає три числа a, b і c та повертає найбільше з них.
Використовуйте цю функцію для знаходження найбільшого з трьох введених користувачем чисел та виведіть результат.

Завдання 3: Перетворення градусів Цельсія у Фаренгейти
Напишіть функцію celsius_to_fahrenheit, яка приймає один параметр - температуру в градусах Цельсія.
Функція повинна перетворювати температуру у Фаренгейти та повертати результат.
Викличте функцію для введеного користувачем значення та виведіть результат.

Завдання 4: Обчислення середнього арифметичного
Напишіть функцію calculate_average, яка приймає один параметр - список чисел.
Функція повинна обчислювати та повертати середнє арифметичне цих чисел.
Викличте функцію для введеного користувачем списку чисел та виведіть результат.

Завдання 5: Підрахунок голосних
Напишіть функцію count_vowels, яка приймає один параметр - рядок.
Функція повинна підраховувати та повертати кількість голосних літер у цьому рядку.
Викличте функцію для введеного користувачем рядка та виведіть результат.
