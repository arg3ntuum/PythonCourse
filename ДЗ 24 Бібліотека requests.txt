Завдання 1: Простий GET запит
Напишіть програму, яка за допомогою бібліотеки requests відправляє GET запит до сайту, наприклад, https://jsonplaceholder.typicode.com/posts, і виводить перші 5 записів на екран.

Завдання 2: Відправка даних методом POST
Напишіть програму, яка відправляє POST запит на той самий сайт, додаючи новий пост. Використовуйте дані:

{
    "title": "foo",
    "body": "bar",
    "userId": 1
}
Виведіть відповідь сервера на екран.

Завдання 3: Обробка помилок
Напишіть програму, яка робить GET запит до неіснуючого ресурсу, обробляючи можливі помилки. Виведіть відповідний код помилки, якщо запит не вдалося виконати.

Завдання 4: Отримання даних з JSON
Напишіть програму, яка робить GET запит до API (наприклад, https://api.github.com/users/octocat), отримує відповідь у форматі JSON і виводить ім'я користувача та кількість репозиторіїв.

Завдання 5: Скачування файлу
Напишіть програму, яка за допомогою бібліотеки requests завантажить зображення з веб-сайту і збереже його на локальний диск.
