Завдання 1: Простий HTTP GET запит
Напишіть програму, яка надсилає HTTP GET запит на будь-яку веб-сторінку (наприклад, https://jsonplaceholder.typicode.com/posts) і виводить на екран статусний код та вміст сторінки.

Завдання 2: HTTP POST запит
Напишіть програму, яка надсилає HTTP POST запит на сервер (наприклад, https://jsonplaceholder.typicode.com/posts) із даними у форматі JSON. Виведіть на екран відповідь сервера.

Завдання 3: Отримання даних за допомогою параметрів запиту
Напишіть програму, яка надсилає GET запит до сервера з параметром запиту. Наприклад, використовуйте https://jsonplaceholder.typicode.com/posts/1, де параметр - ID ресурсу, і виведіть отриману інформацію.

Завдання 4: Обробка помилок у HTTP запитах
Створіть програму, яка відправляє GET запит на неіснуючий URL. Обробіть можливі помилки (наприклад, 404 або 500) і виведіть відповідне повідомлення про помилку.

Завдання 5: HTTP запит з заголовками
Напишіть програму, яка відправляє GET запит на будь-який сервер із власними заголовками (наприклад, User-Agent) та виводить на екран результат запиту.
