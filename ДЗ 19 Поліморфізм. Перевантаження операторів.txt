Завдання 1: Поліморфізм в Python
Створіть клас Animal, в якому є метод sound(), що виводить звук, який робить тварина. Потім створіть два підкласи: Dog і Cat. Вони повинні перевизначати метод sound() для виведення власних звуків: для собаки — "Woof", для кота — "Meow".

Завдання 2: Перевантаження операторів
Створіть клас Rectangle, що має атрибути для ширини та висоти. Перевантажте оператор + для цього класу, щоб можна було додавати два прямокутники. В результаті додавання прямокутників повинні обчислюватися нові ширина і висота, як сума відповідних атрибутів.

Завдання 3: Поліморфізм із різними видами тварин
Розширіть клас Animal, додаючи метод move(). У класі Dog метод move() повинен виводити "Running", а в класі Cat — "Jumping". Створіть список з об'єктів цих класів і викличте метод move() для кожного об'єкта.

Завдання 4: Перевантаження оператора порівняння
Створіть клас Book, що має атрибути для назви та кількості сторінок. Перевантажте оператор == для порівняння двох книг за кількістю сторінок. Якщо кількість сторінок однакова, об'єкти повинні вважатися рівними.

Завдання 5: Поліморфізм в комбінованих класах
Створіть клас Shape з методом area(), який буде обчислювати площу. Створіть два підкласи: Circle та Square, які повинні перевизначати метод area() для обчислення площі кола та квадрата відповідно. Створіть список з об'єктами цих класів і виведіть площу кожної фігури.
